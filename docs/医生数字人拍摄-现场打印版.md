# 数字人拍摄 · 现场速查卡（打印贴在拍摄位旁）

> 详细说明见《医生数字人拍摄指南》。本页用于现场快速核对。

---

## ① 开拍前 · 布场检查

- [ ] **三脚架**架稳，镜头与医生**视线齐平**（不俯不仰）
- [ ] **左右补光**，脸打亮、无阴影；**背后无窗、不逆光**
- [ ] 背景**纯色干净**，无杂物、无人走动
- [ ] **半身入镜**（胸部以上），脸居中，别裁太紧
- [ ] 医生穿**纯色衣服**（无细密条纹/格子/反光）
- [ ] 设备：电量足、存储够、对焦准、收音正常
- [ ] **已和医生商量好一个他熟悉的话题**（专业或生活均可）

## ② 话题方法 ·【严禁朗读稿】

> **绝对不要让医生念稿、背词。** 让他**像跟对面一个人聊天**那样，
> 围绕自己熟悉的话题自然地讲，**讲什么不重要，状态自然才重要。**
> 开机前先闲聊两句帮他放松，卡壳了就提醒"接着聊"，别停。
> **说错了也没关系，直接接着说，不用重录！**（内容对错完全不影响效果）

## ③ 录制中 · 提醒医生

- [ ] **一镜到底**，中途不停、不剪
- [ ] 连续讲 **2~5 分钟**（奔 5 分钟拍）
- [ ] **全程直视镜头**
- [ ] 开头先**停 2~3 秒**再开口；长句间停 1~2 秒，**停顿时闭嘴**
- [ ] 坐/站姿稳，**不晃身、不甩头**，自然呼吸
- [ ] 手势（如有）控制在**胸部以下**

## ④ 别忘了 · 授权视频（Consent）

- [ ] 同一人、同一环境、同样清晰光线
- [ ] 医生对镜头**照 HeyGen 屏幕提示**清晰念出同意声明

## ⑤ 收设备前 · 当场回放

- [ ] 训练视频：清晰(≥1080p)、不抖、不暗、声音清楚、≥2 分钟
- [ ] 授权视频：清晰、同一人
- [ ] 两段都**回放确认**、文件**完整保存**
