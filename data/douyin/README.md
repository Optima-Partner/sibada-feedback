## 目录结构

每个账号目录下两个文件：

- `profile.json` — 主页信息：uid、secUid、账号名、粉丝数、视频总数、总获赞、简介等
- `videos.json` — 全量视频元数据（按发布时间倒序）：视频 id、分享链接、描述、点赞/评论/分享/收藏数、话题标签、发布时间、时长、临时播放地址

`crawl-summary.json` 记录本次抓取的账号清单与完整性核对（videos_fetched 与 profile 的 totalVideos 全部一致）。
