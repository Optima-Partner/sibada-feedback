# sibada-feedback

本仓库用于收集 **sibada-platform** 项目的反馈，包括 Bug 报告、功能建议和使用问题。

## 如何提交反馈

请通过 [Issues](https://github.com/Optima-Partner/sibada-feedback/issues) 提交反馈：

1. 点击 **New issue**
2. 尽量提供清晰的标题和描述

### 提交 Bug 时建议包含

- **复现步骤**：如何触发这个问题
- **预期结果** 与 **实际结果**
- **环境信息**：环境（dev / stage / prod）、浏览器/客户端版本、相关时间点
- **截图或日志**（如有）

### 提交功能建议时建议包含

- **要解决的问题 / 场景**
- **期望的功能或行为**
- **优先级或影响范围**（可选）

## 说明

- 本仓库仅用于反馈收集，**不包含 sibada-platform 源代码**。
- 提交前请先搜索已有 Issue，避免重复。
