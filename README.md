## 🐞 如何提交反馈

点击上方绿色按钮（或 [这里](https://github.com/Optima-Partner/sibada-feedback/issues/new)）即可进入新建反馈页面。填写时建议尽量说明：
- **遇到的问题 / 期望的功能**
- **复现步骤**（如果是 Bug）
- **截图或录屏**（可选，能帮助我们更快定位）
- **使用环境**（环境 dev / stage / prod、浏览器或客户端版本、相关时间点）

提交后我们会及时查看并回复，感谢你帮助斯巴达变得更好 🙏

## 🏥 关于斯巴达互联网医院

**斯巴达互联网医院 —— 辅助生殖 AI 智慧平台**

斯巴达是一个面向辅助生殖领域的 AI 智慧医疗平台，借助 AI 数字人医生与智能体能力，为患者提供更便捷、更有温度的就医与咨询体验，并辅助医生完成科普、随访等工作。

平台核心能力：

| 能力 | 说明 |
| --- | --- |
| 🧑‍⚕️ AI 数字人医生 | 形象与声音克隆，提供拟真的医患交互体验 |
| 💬 智能问诊咨询 | 基于大模型的辅助生殖知识问答与导诊 |
| 🎬 数字人视频生成 | 自动生成科普 / 随访视频内容 |
| 📚 专业知识库 | 沉淀辅助生殖领域的专业内容与诊疗经验 |

**核心价值**：
- 降低优质医疗资源的获取门槛
- 提升科普与随访的效率
- 为患者提供 7×24 小时的智能咨询陪伴
